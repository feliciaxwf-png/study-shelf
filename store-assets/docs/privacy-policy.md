# Privacy Policy for Study Shelf

Effective date: April 27, 2026

Study Shelf is a Chrome extension that helps users organize open tabs into a learning list.

## Data the extension handles

Study Shelf may access:
- The titles and URLs of currently open browser tabs
- Page metadata used to suggest tags when the user saves a page
- User-entered learning status, tags, and key takeaways

## How data is used

This data is used only to provide the extension's core features:
- Grouping open tabs by website
- Saving pages into a local learning list
- Letting users search, tag, filter, and reopen saved pages
- Closing tabs when the user chooses to close them

## Data storage

Study Shelf stores saved pages and learning notes locally using Chrome storage. The extension does not operate a remote server for this data.

## Data sharing

Study Shelf does not sell user data. Study Shelf does not share saved pages, notes, browsing activity, or learning data with third parties.

## Remote code and ads

Study Shelf does not execute remote code and does not display ads.

## Permissions

Study Shelf requests Chrome permissions only for its learning workflow:
- `tabs`: to read open tab titles and URLs, focus tabs, create tabs, and close tabs when requested by the user
- `storage`: to save the user's learning list locally
- `scripting`: to read page metadata only when the user saves a page
- host permissions: to allow metadata extraction from pages the user chooses to save

## Contact

For privacy questions, use the support contact listed on the Chrome Web Store listing.
