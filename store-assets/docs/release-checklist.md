# Release Checklist

## Before upload
- Reload the unpacked extension in `chrome://extensions`.
- Open a new tab and test English and Chinese.
- Test Save, Save group, Close, Remove, Search, Status, Tags, and Key takeaway.
- Confirm the left/right resizer works.
- Confirm the extension icon appears in `chrome://extensions`.
- Confirm there are no console errors.

## Chrome Developer Dashboard
- Upload `study-shelf-release.zip`.
- Fill Store listing using `store-listing.md`.
- Upload screenshots from `store-assets/screenshots`.
- Fill Privacy practices using `privacy-policy.md` and `permission-justification.md`.
- Choose category: Productivity.
- Choose distribution: Public or trusted testers first.

## Suggested first release strategy
Publish first to trusted testers if you want feedback before public launch. Use public release after you are comfortable with the install flow and privacy review text.
